## 🚀 快速使用指南

### 安装方法

```powershell
# 1. 以管理员身份运行 PowerShell

# 2. 设置执行策略（如需要）
Set-ExecutionPolicy Bypass -Scope Process -Force

# 3. 运行安装脚本
.\SecurityTools_Installer.ps1

# 4. 输入 "I ACCEPT" 接受法律条款

# 5. 等待自动下载安装完成（约15-20分钟）
```

已配置的命令别名

安装完成后，以下命令别名自动添加到系统PATH：

```bash
# 信息收集
nmap -sV 192.168.1.1           # 端口扫描
shodan host 8.8.8.8            # 查询IP信息

# Web测试
sqlmap -u "http://target.com?id=1"  # SQL注入测试
burpsuite                           # 启动Burp Suite

# 密码工具
hashcat -m 0 hash.txt rockyou.txt   # 密码破解
aircrack-ng capture.cap             # 无线破解

# 防御监控
wireshark                           # 启动Wireshark
snort -i eth0 -c snort.conf         # 启动Snort
```

❓ 常见问题

Q: 提示"无法加载脚本"怎么办？

```powershell
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
```

Q: 部分工具安装失败？

· 检查网络连接
· 某些工具需要手动下载（如 Nessus、Burp Suite）
· 查看日志文件：$env:USERPROFILE\Desktop\SecurityTools\install.log

Q: winget 不可用？

从 Microsoft Store 安装"应用安装程序"即可获得 winget 支持。

Q: 如何卸载工具？

```powershell
# 使用 winget 卸载
winget uninstall --id <工具ID>

# 或手动删除安装目录
```
